# Beginner Cybersecurity Projects

This repository contains a collection of beginner-friendly cybersecurity projects completed in a controlled lab environment using Kali Linux. These projects were designed to develop foundational skills in network scanning, traffic analysis, web application testing, password security and Linux system hardening.

Each project folder includes a detailed `notes.txt` file outlining the steps taken, observations and recommendations. Relevant screenshots and files generated during the exercises are also included where applicable.
